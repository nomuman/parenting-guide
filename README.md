# Parenting Guide

出産準備から子育てまでをサポートする総合ガイドリポジトリです。  
初めての育児に必要な情報を体系的にまとめています。

## 特徴
- 出産準備から新生児ケアまで網羅
- パパ向けサポートガイド付き
- 便利なアプリ・サービス情報を掲載
- 緊急時の対応マニュアルを提供
- 育児関連書籍のレビューと学びの記録

# 赤ちゃんの睡眠ガイド

## 1. 新生児期（0～3ヶ月）
### 睡眠パターン
- 1日16～18時間
- 2～4時間ごとに起きる
- **根拠**: アメリカ睡眠医学会ガイドライン

### 環境作り
- 室温：20～23℃
- 湿度：50～60%
- 明るさ：薄暗く
- 音：ホワイトノイズが有効

## 2. 乳児期（4～11ヶ月）
### 睡眠リズム
- 1日12～15時間
- 昼寝：2～3回
- 夜間：6～8時間連続

### 寝かしつけのコツ
- 就寝ルーティンを作る
- 入眠儀式を習慣化
- 自己鎮静を促す

## 3. 安全対策
### SIDS予防
- 仰向け寝を徹底
- ベッド周りを安全に
- タバコの煙を避ける
- **根拠**: 日本小児科学会提言

# 妊娠中期から2歳までの詳細ガイド

## 1. 妊娠中期（16～27週）

### 1.1 週別母体の変化
#### 16週
- **身体変化**:
  - 子宮の大きさ：握りこぶし大
  - 体重増加：1.5～2kg
  - つわりが軽減
  - **根拠**: 日本産科婦人科学会「妊娠週数別変化」

- **必要なケア**:
  - 妊婦健診：4週間に1回
  - 体重測定：毎週
  - 血圧チェック
  - **根拠**: 厚生労働省「妊婦健康診査の手引き」

#### 17週
- **身体変化**:
  - 子宮底長：15cm
  - 乳房の張りが増す
  - おりものが増加
  - **根拠**: 日本産婦人科医会

- **必要なケア**:
  - マタニティ下着の着用
  - 保湿ケアの開始
  - 水分補給：1日2リットル
  - **根拠**: 日本皮膚科学会

#### 18週
- **身体変化**:
  - 子宮の大きさ：ソフトボール大
  - 胎動を感じ始める可能性
  - 腰痛が始まる
  - **根拠**: 日本周産期・新生児医学会

- **必要なケア**:
  - 正しい姿勢の維持
  - 軽いストレッチ
  - 腰痛予防ベルトの使用
  - **根拠**: 日本整形外科学会

#### 19週
- **身体変化**:
  - 子宮底長：18cm
  - お腹の膨らみが目立ち始める
  - 便秘がちになる

- **必要なケア**:

  - 適度な運動：1日30分歩行
  - 水分補給の徹底
